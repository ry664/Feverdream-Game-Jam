# Feverdream Game Jam
Being made a a submision for [Fever Dream Game Jam](https://itch.io/jam/feverdream-game-jam3)

an ultrakill-like fast paced shooter within a small dense open world
aquire upgrades through combat, light puzzles, and deduction from npcs

A simple 3d World with 2d billboard objects and chreacters simmilar to [Ena Dream BBQ](https://store.steampowered.com/app/2134320/ENA_Dream_BBQ/)
explore 3+ themed areas and find their secrets
