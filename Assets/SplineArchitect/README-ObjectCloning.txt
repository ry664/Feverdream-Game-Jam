README-ObjectCloning.txt
================================
Object Cloning is an addon for Spline Architect that is included in the base package.
It allows you to automatically fill splines with GameObjects or meshes.

In the SplineArchitect main folder you have:
1. 'User manual - Addons.pdf' for setup and usage instructions.
2. Look at scenes MeshDeformation, BasicTrafficSystem or BasicTrainSystem in the "Example" folder for usage demonstrations.
